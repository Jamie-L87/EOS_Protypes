EOS Design System

Style Guide

Accessibility — Global

https://www.w3.org/TR/WCAG21/

Highlights

  • Development & design adheres to WCAG 2.1

    All components follow Web Content Accessibility Guidelines 2.1 standards

  • Ensure minimum color contrast ratios

    Maintain AAA level contrast ratios where possible for readability

  • Utilizing 44px min target states (AAA) where possible

    Touch targets should be at least 44x44px for accessibility

  • All focus states should also include default browser focus

    Keyboard navigation must be clearly visible with default browser focus indicators
